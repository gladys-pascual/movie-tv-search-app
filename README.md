# showme.cf

## App Features

- Search movies or tv shows
- Login to your own account
- Mark and view favourite movies or tv shows
- Bookmark search results, movies and tv shows
- Responsive design

TMDb API was used to create this project.

## Technologies used:

- JavaScript, React
- HTML
- CSS, SASS, Flexbox

## Libraries used:

- react
- react-router-dom
- node-sass
- react-animated-heart
- react-rater
- react-spinners
